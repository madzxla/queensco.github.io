#2 - XenPOS User Interface

Watch the full demo video here or try it out yourself

Scripting by nat1xone

XenPOS is a Cafe POS system for ro-restaurants and cafes. The user interface is modular and organized to fit any game. You can easily edit the interface to match your needs; no UI knowledge is necessary. The system was scripted by nat1xone using Fusion, a reactive Roblox UI framework. It uses client-side UI prediction to ensure stable performance on all networks.

#4 - Clocked In UI Concept

Watch the full demo video here or try it out yourself

In Clocked In, players can roleplay as a floor worker in a retail store. Your goal is to work together with others to keep the store running and serve an infinite number of NPC customers. I took inspiration from the Roblox Xbox UI for the button effect, and it fits very nicely everywhere, not only in console environments. I was also tasked with scripting the animations for the UI.

How much do you charge?

Great question! I'm one of the few UI/UX developers in the industry that do not charge per frame, meaning your price will be adjusted to the difficulty and the requirements of your order, meaning something as simple as a box with text and a few buttons will not come up to 5k Robux, just because it's a flat rate. You should contact me to get an estimate for your order.

Can I request revisions?

You're free to request as many revisions as you want, I don't set a limit and I won't charge you extra for them as long as it's reasonable and doesn't include swapping the core style or scrapping my current work.

Can you script my UI?

I don't offer these services. There are many better dedicated programmers who focus on this field who could write a lot better code than me. I will not take requests to code the UI, so you should find someone to code it prior to submitting the order or coding it yourself, there are many great resources available on the DevForum and YouTube.
